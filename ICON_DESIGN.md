# Plugin Icons - Nord Recharged

## Design Concept

The Nord Recharged plugin icon features a bold letter **"R"** in a circular badge, following Nord's design philosophy.

### Color Palette

**Light Theme Icon (pluginIcon.svg):**
- Background: Nord Frost gradient (#5E81AC → #88C0D0)
- Letter: Snow Storm gradient (#ECEFF4 → #D8DEE9)
- Style: Clean, modern, bright

**Dark Theme Icon (pluginIcon_dark.svg):**
- Background: Polar Night gradient (#2E3440 → #4C566A)
- Letter: Frost cyan gradient (#88C0D0 → #81A1C1)
- Style: High contrast, glowing effect

### Design Elements

1. **Circular Badge**: 40x40px SVG, scalable
2. **Letter "R"**: Bold, sans-serif style
   - Vertical stem
   - Curved bowl
   - Diagonal leg (rotated 25°)
3. **Subtle Accents**:
   - Inner circle glow (0.5px stroke, 20-30% opacity)
   - Outer rim accent for depth
4. **Gradients**: Smooth transitions using Nord official colors

### Usage

- `pluginIcon.svg` - Used in light UI themes
- `pluginIcon_dark.svg` - Used in dark UI themes
- Both located in `resources/META-INF/`
- Automatically picked by IntelliJ based on UI theme

### Inspiration

Inspired by the official Nord theme logo but customized with "R" for "Recharged", maintaining the Arctic, minimal aesthetic.
